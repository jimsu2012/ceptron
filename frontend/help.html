COMMANDS
img { keywords of image }
Generates an image based on a description
ex: img bird flying in the sky
sum { url } or sum { 10 } { url }
Generates an automatic summary of the url in 10 sentences by default
ex: sum https://en.wikipedia.org/wiki/Perceptron
sum { n } { url }
Generates an automatic summary of the url in n sentences
ex: sum 3 https://en.wikipedia.org/wiki/Perceptron
HOTKEY CONTROLS
Ctrl + Shift + C [Windows] or Cmd + Shift + C [Mac]
Opens command field
esc key
Exits command field
Enter key
Executes typed command
