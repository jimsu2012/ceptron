COMMANDS
image { keywords of image }
Generates an image based on a description
ex: image bird flying in the sky
summarize { url } or summarize { n } { url }
Generates an automatic summary of the url in 10 sentences by default
ex: summarize 3 https://en.wikipedia.org/wiki/Perceptron
instruct { n } { prompt }
Generates a text with length based on n that answers the prompt provided (default 200)
ex: instruct 300 Why should we host a hackathon?
history
Opens history menu containing previous 8 commands
help
Opens help menu containing commands and hotkeys
prune
Clears output cache
clear
Clears command history
HOTKEY CONTROLS
Ctrl + Shift + C [Windows] or Cmd + Shift + C [Mac]
Opens command field
esc
Exits command field
Enter
Executes typed command
Up Arrow
Gets previous command
Down Arrow
Gets next command
